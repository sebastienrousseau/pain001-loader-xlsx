## Executive Summary

Initial release of `pain001-loader-xlsx`, a third-party loader
plugin that teaches the [`pain001`](https://github.com/sebastienrousseau/pain001)
ISO 20022 payment library to read payment data directly from Excel
`.xlsx` / `.xlsm` files. **No "Save As CSV" step.** Drop-in:
install both packages and `.xlsx` files dispatch automatically.

This package is the canonical worked example of pain001's v0.0.54+
plugin substrate — the same `AbstractLoader` Protocol any
third-party loader uses.

## What's in v0.0.53 at a glance

- **`XlsxLoader`** — implements the structural
  `pain001.plugins.AbstractLoader` Protocol (no subclassing).
- **First-sheet dispatch** with `data_only=True` so formulas
  resolve to their cached last-saved value.
- **Streaming variant** for batches that don't fit in memory.
- **IBAN safety guard** — refuses numeric-typed IBAN cells with a
  clear remediation (the leading-zero-stripping pattern that
  breaks SAP / Oracle / Workday exports).
- **Entry-point auto-discovery** via the standard `pain001.loaders`
  group; pain001 picks up the loader at process start with no
  manual wiring.
- **Two runnable examples** at `examples/` exercised by CI.

## Use the new plugin

### CLI (auto-dispatched)

```bash
pip install pain001 pain001-loader-xlsx

pain001 -t pain.001.001.03 -d payments.xlsx -o out.xml
# pain001's universal loader dispatches .xlsx through this plugin
# automatically; XML is XSD-validated before write.
```

### Confirm pain001 sees it

```bash
pain001 plugins list --kind loader
# loader  xlsx  pain001-loader-xlsx==0.0.53  Read flat-record ...
```

### From Python

```python
from pain001_loader_xlsx import XlsxLoader

loader = XlsxLoader()
result = loader.load("payments.xlsx")
print(result.source_hint, len(result.rows))
# -> "payments.xlsx" 42
```

## Requirements

- Python 3.10+.
- `pain001 >= 0.0.54, < 1` — the plugin substrate
  (`pain001.plugins`) lands in pain001 v0.0.54. The package metadata
  declares this dependency explicitly; `pip` will pull in a
  compatible pain001 automatically.
- `openpyxl >= 3.1, < 4`.
